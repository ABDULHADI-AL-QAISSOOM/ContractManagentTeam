How to use the Tool

This tool compares data in the Contract Management File against the Active Employee List. Discrepancies are flagged if the Contract file doesn't match the Source of Truth.

Validated Columns:
  • PERSON_NUMBER (Key)
  • NATIONALITY
  • WORKER_TYPE
  • Age_H
  • AGE_HIJRI (Absolute)
  • JOB_NAME
  • CONTRACT_END_DATE
  • PERFORMANCE_EVALUATION_TYPE
  • 2024_Evaluation
  • 2025_Evaluation
